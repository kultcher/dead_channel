To play:
Make sure that null_signal.exe and null_signal.pck are in the same directory and run null_signal.exe

Windows will probably throw a warning but I promise there are no viruses. Just hit "more info" and "run anyway"

Notes:
- There's no audio yet.
- Hit F1 for in-game help.
- The game has a couple pieces of placeholder art that is AI-generated. Not intended for the full game.

Known issues:
- The game is built for 1920x1080 resolution, I haven't tested it in other resolutions but it may break.
- It's possible in a couple places to sequence break and cause the game to stop progressing. Mostly happens in the first half if you try to do things ahead of the tutorial prompting.
